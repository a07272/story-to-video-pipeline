如何重現（概要）

  1. 把小說段落交給 LLM，生成 scene-by-scene 分鏡（包含旁白、台詞與時間長度）。
  2. 用圖像模型（Stable Diffusion / gpt-image-1）產生角色與場景圖片（多角度、表情）。li>
  3. （可選）使用 DreamBooth/LoRA 建立角色一致性模型。
  4. 用 TTS 生成旁白音軌（ElevenLabs / Google TTS），再以 FFmpeg 合成音軌與圖片成短 clip。
  5. 把短 clip 合併成一集（使用 FFmpeg 或 Pika/Runway API），上傳至 YouTube，並把影片連結放回本 repo 的 pages。

注意：任何上傳的素材請確認著作權與模型授權。更多細節請參考 README。
