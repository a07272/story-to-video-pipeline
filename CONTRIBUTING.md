# Contributing

感謝你有興趣參與本專案！以下為快速指引。

## 回報 Bug 或 建議 Feature
1. 在本 repo 的 Issues 裡建立新的 issue，選擇適當的 template（Bug / Enhancement / Prompt Improvement）。

## 提交 PR
1. Fork 本 repo，建立 feature 分支（ex: `feat/prompt-improve`）。
2. 提交程式或文件修改，包含清楚的 commit message。
3. 開 PR 並描述你做了什麼、為何需要、如何測試。

## Prompt / Workflow 提交
- 在 `prompts/` 新增或修改 `.md` 檔，包含 example input/output 與模型測試結果說明。
- 在 `workflows/` 提交工作流 JSON 範例（請勿包含實際的 API Key）。

## 大型媒體上傳
- 請勿直接把 >50MB 的影片上傳到 repo。請上傳至 YouTube（unlisted/private）或 S3，並在 README 中提供連結與授權說明。

## 行為守則
- 尊重他人、避免發佈侵權或有害內容。
- 若發現敏感/違法內容，請透過 issue 私下通知 maintainer。

感謝你的貢獻！
